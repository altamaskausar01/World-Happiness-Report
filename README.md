🌍 World Happiness Index Report Analysis

📖 Project Overview

The World Happiness Index Report Analysis project explores global happiness trends using the World Happiness Report dataset. The project compares happiness scores of India, Finland, and the World Average over multiple years and analyzes factors that influence happiness such as GDP, social support, health, freedom, generosity, and corruption perception.

The project uses Python for data cleaning, analysis, and visualization to uncover meaningful insights from the dataset.

---

🎯 Objectives

 Analyze global happiness trends over time.
 Compare happiness scores between India, Finland, and the World Average.
 Identify relationships between happiness and socioeconomic factors.
 Visualize data using various chart types.
 Understand key contributors to happiness across countries.

🔍 Key Findings

 Finland consistently records higher happiness scores compared to the global average.
 India demonstrates gradual changes in happiness levels over the years.
 Economic strength, healthcare, social support, and freedom significantly impact happiness.
 Social and economic indicators strongly correlate with Life Ladder scores.
